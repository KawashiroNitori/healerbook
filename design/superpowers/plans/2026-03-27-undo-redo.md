# 撤销/重做功能实现计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 为时间轴编辑器的所有编辑操作添加撤销/重做功能

**Architecture:** 使用 zundo 中间件包装 timelineStore，通过 `partialize` 只跟踪 `timeline` 字段。使用 react-hotkeys-hook 统一管理快捷键（Mod+Z / Mod+Shift+Z），同时迁移现有的 Delete/Backspace 快捷键。在 EditorToolbar 添加撤销/重做按钮。

**Tech Stack:** zundo, react-hotkeys-hook, Zustand, React, Lucide React, Vitest
