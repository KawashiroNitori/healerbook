## 1. 如何新建空白时间轴

当你想从头规划一场副本的减伤，而不是从 FFLogs 导入时：

1. 在首页点击 **「新建时间轴」**
2. 填写 **时间轴名称**（比如 "M12S 固定队减伤 v1"）
3. 在 **「副本」** 下拉菜单中选择你要规划的副本
   - 副本按版本分组，比如「阿卡狄亚零式 重量级 (7.4)」下有 M9S ~ M12S
4. 点击 **「创建」**

<!-- 📸 截图：新建时间轴对话框
     内容：CreateTimelineDialog 已填写名称，副本下拉菜单展开状态，
     可见按版本分组的副本列表。 -->

创建后会在新标签页打开编辑器。时间轴是空的，你需要自己添加伤害事件和减伤技能。

> 目前支持的副本：M9S ~ M12S（含二形态）和 FRU 光暗未来绝境战。如果选择了尚未支持的副本，顶部会出现黄色提示条。

## 2. 如何从 FFLogs 导入战斗数据

从 FFLogs 导入可以自动获取一场战斗中的所有伤害事件和减伤技能使用记录。

### 准备工作

在 FFLogs 网站上找到你想导入的战斗报告，选择一场具体的战斗（boss），然后复制浏览器地址栏的完整链接。

链接格式举例：

```
https://www.fflogs.com/reports/ABC123#fight=5
```

### 导入步骤

1. 在首页点击 **「从 FFLogs 导入」**
2. 粘贴 FFLogs 链接到输入框（打开对话框时会自动尝试读取剪贴板）
3. 检查输入框下方是否正确显示了 **报告代码** 和 **战斗编号**
4. 点击 **「导入」**
5. 等待导入完成（通常几秒钟）

<!-- 🎬 动图：完整导入流程
     内容：点击「从 FFLogs 导入」→ 对话框弹出 → 粘贴链接 →
     下方显示解析结果 → 点击「导入」→ 显示进度（获取报告信息...获取战斗事件...解析数据...）→
     自动跳转到编辑器页面。时长 8-12 秒。 -->

### 导入后会得到什么

- **小队阵容**：自动识别参战的 8 名玩家及其职业
- **伤害事件**：该场战斗中所有 AOE 和死刑，含原始伤害值、伤害类型
- **减伤技能**：每个人实际使用的减伤技能及时间点
- **回放模式**：导入后默认进入回放模式，显示实战中每个人的实际受伤数据

### 常见问题

| 问题             | 原因及解决                                                             |
| ---------------- | ---------------------------------------------------------------------- |
| 提示缺少战斗编号 | 链接里没有 `#fight=数字`，请在 FFLogs 上选择具体的某场战斗后再复制链接 |
| 导入失败         | 可能是报告设为私密（需要报告上传者设为公开），或网络问题               |

## 3. 如何管理小队阵容

阵容决定了时间轴上显示哪些职业的技能轨道，以及有哪些减伤技能可用。

### 查看和快速操作

点击工具栏上的 **「小队阵容 X/8」** 按钮，弹出阵容面板：

- 每个队员显示 **职业图标** 和 **职业名称**
- 点击队员旁边的 **眼睛图标**：隐藏/显示该玩家的技能轨道
- **Shift + 点击眼睛图标**：只显示这一个人，隐藏其他所有人（方便专注看某个职业的减伤安排）
- 点击 **X 图标**：移除该队员

<!-- 📸 截图：阵容弹出面板
     内容：CompositionPopover 展开状态，显示 8 名队员列表，
     每人有职业图标、名称、眼睛按钮和 X 按钮，底部有「调整阵容」按钮。 -->

<!-- 🎬 动图：Shift+点击独显玩家
     内容：阵容面板中 Shift+点击白魔的眼睛图标 → 时间轴上其他职业轨道消失，
     只剩白魔一行 → 再次 Shift+点击恢复全部显示。时长 4-6 秒。 -->

### 编辑完整阵容

1. 在阵容面板底部点击 **「调整阵容」**
2. 在弹出的对话框中：
   - **上方**：显示当前阵容，点击职业图标可移除
   - **下方**：按角色分组（坦克 / 治疗 / 近战 / 远敏 / 法系）显示所有可选职业，点击添加
3. 阵容最多 8 人，满员后无法继续添加
4. 点击 **「完成」** 保存

<!-- 📸 截图：调整阵容对话框
     内容：CompositionDialog 打开状态，上方显示当前 8 人阵容图标，
     下方按角色分组排列所有可选职业图标（部分已置灰表示满员不可添加）。 -->

## 4. 如何添加和编辑伤害事件

伤害事件代表副本中 boss 的攻击，是减伤规划的基础。

### 添加伤害事件

**方法一：双击**

- 在伤害事件轨道的空白区域**双击鼠标左键**，会弹出添加对话框，时间自动填为你双击位置对应的秒数

**方法二：右键菜单**

- 在伤害事件轨道空白处**右键**，选择 **「添加」**

<!-- 📸 截图：右键菜单
     内容：在伤害事件轨道空白处右键弹出的上下文菜单，
     显示「添加」「粘贴」「添加注释」等选项。 -->

### 填写事件信息

| 字段       | 说明                                                                  |
| ---------- | --------------------------------------------------------------------- |
| 事件名称   | 给这次攻击起个名字，比如 "P2 暗天之火"                                |
| 时间（秒） | 从战斗开始算起的秒数，支持小数（精度 0.1 秒），也可以填负数表示开怪前 |
| 伤害值     | 未减伤的原始伤害数值                                                  |
| 攻击类型   | **AOE**（全体攻击）或 **死刑**（坦克单吃）                            |
| 伤害类型   | **物理** / **魔法** / **特殊**（影响牵制和昏乱的生效判定）            |

### 编辑已有事件

1. **单击**伤害事件方块将其选中
2. 右侧属性面板会显示所有字段，直接修改即可
3. 修改实时生效，不需要额外保存

<!-- 📸 截图：属性面板编辑状态
     内容：选中一个伤害事件后的属性面板，显示事件名称输入框、时间/伤害数字输入框、
     伤害类型和攻击类型下拉菜单，所有字段可编辑。 -->

### 调整事件时间

直接**拖拽**伤害事件方块左右移动。

<!-- 🎬 动图：拖拽伤害事件
     内容：鼠标按住一个蓝色 AOE 方块，左右拖动到新位置后松手。时长 3-4 秒。 -->

### 复制和粘贴事件

- 选中事件后按 **Ctrl + C** 复制
- 按 **Ctrl + V** 粘贴到当前光标位置
- 也可以右键选择 **「复制」** 和 **「粘贴」**

> 复制文本功能（右键→「复制文本」）会把事件信息以文字形式复制到剪贴板，方便发到聊天软件里。

### 删除事件

- 选中后按 **Delete** 或 **Backspace** 键
- 或右键选择 **「删除」**
- 也可以在属性面板右上角点击垃圾桶图标

## 5. 如何放置和调整减伤技能

### 放置技能

**方法一：双击**

- 在目标职业的技能轨道空白处**双击鼠标左键**，在该时间点添加当前选中的减伤技能

**方法二：右键菜单**

- 在技能轨道空白处**右键**，选择 **「添加」**

> 如果提示"该技能与已有技能重叠"，说明同一个技能在 CD 时间内被放了两次，需要调整时间间隔。

### 调整技能时间

直接**拖拽**技能图标左右移动到合适的位置。

### 查看技能详情

**悬浮**鼠标到技能图标上，会显示技能提示，包含：

- 技能名称和图标
- 咏唱时间、复唱时间（CD）
- 效果范围
- 技能效果描述
- 适用职业

**单击**技能图标可以固定提示框，方便阅读。

<!-- 📸 截图：技能悬浮提示
     内容：鼠标悬浮在节制图标上，弹出的 ActionTooltip 显示技能图标、
     名称、咏唱/复唱时间、效果描述、适用职业图标等完整信息。 -->

### 删除技能

- 选中后按 **Delete** 键
- 或右键选择 **「删除」**

## 6. 如何阅读减伤效果预估

选中一个 **AOE** 类型的伤害事件后，右侧属性面板会显示减伤效果预估。

> 死刑类事件不会显示预估，因为死刑伤害由坦克单独承受，不参与全队减伤计算。面板会提示"死刑不参与团减计算"。

### HP 条

最直观的指标。显示这次 AOE 后全队（非坦克）的大致血量：

- **绿色** = 剩余 HP
- **红色条纹** = 受到的伤害
- 文字格式：`剩余HP / 最大HP (-伤害量)`

<!-- 📸 标注图：属性面板减伤效果完整解读
     内容：属性面板中一个有充分减伤的 AOE 事件的完整截图，用箭头/编号标注：
     ① HP 条（绿色/红色区段）
     ② HP 数值文字
     ③ 减伤构成条（蓝/黄/红各色段）
     ④ 百分比减伤状态图标列表及合计值
     ⑤ 盾值状态图标列表及合计值
     这是操作指南第 6 节的核心配图。 -->

### 警告信号

| 标记                    | 含义                        | 你应该做什么                               |
| ----------------------- | --------------------------- | ------------------------------------------ |
| 💀 **致死**（红色背景） | 伤害超过最大 HP，会团灭     | 必须增加减伤！面板会提示还需要多少额外减伤 |
| ⚠️ **危险**（橙色背景） | 勉强活下来，但剩余 HP < 10% | 建议增加减伤或安排额外治疗，容错很低       |

### 减伤构成条

HP 条下方的彩色横条展示原始伤害的去向：

| 颜色 | 含义                    | 例子                   |
| ---- | ----------------------- | ---------------------- |
| 蓝色 | 被百分比减伤吃掉的伤害  | 雪仇、节制、牵制的效果 |
| 黄色 | 被盾吸收的伤害          | 圣光幕帘、鼓舞的盾值   |
| 红色 | 实际掉的血              | 这才是你需要奶回来的量 |
| 灰色 | 溢出伤害（超出最大 HP） | 只在致死时出现         |

鼠标悬浮在每个色块上可以看到具体数值和百分比。

### 生效状态列表

面板最底部分两组列出生效的减伤：

- **百分比减伤**：每个减伤 Buff 的图标和减伤率，右侧显示合计减伤
- **盾值**：每个盾技能的图标和数值，右侧显示合计盾值

> 小贴士：减伤计算使用乘算公式。比如两个 10% 减伤叠加的效果是 `1 - (1-0.1) × (1-0.1) = 19%`，而不是简单的 20%。所以多层减伤叠在一起的边际收益会递减。

## 7. 如何使用注释

注释可以在时间轴上标记重点信息，比如"这里必须有节制"或"二仇接刀"。

### 添加注释

1. 在伤害事件轨道或技能轨道的空白处**右键**
2. 选择 **「添加注释」**
3. 在弹出的输入框中输入文字（最多 200 字）
4. 按 **Ctrl + Enter** 或点击确认按钮保存

<!-- 🎬 动图：添加和查看注释
     内容：右键技能轨道 → 选择「添加注释」→ 输入 "这里必须有节制" →
     按 Ctrl+Enter 保存 → 时间轴上出现图钉图标 → 悬浮图钉显示注释内容。
     时长 6-8 秒。 -->

### 查看注释

- 注释在时间轴上显示为 **图钉图标**
- **悬浮**鼠标到图钉上：临时显示注释内容
- **单击**图钉：固定显示，再次点击或点击空白处取消固定

### 编辑注释

- **右键**点击图钉图标，选择 **「编辑」**

### 移动注释

直接**拖拽**图钉图标到新的时间位置。

### 删除注释

- 选中后按 **Delete** 键
- 或右键选择 **「删除」**

## 8. 如何发布和分享时间轴

### 前提：登录 FFLogs 账号

1. 点击页面右上角的 **「登录 FFLogs」**
2. 跳转到 FFLogs 网站完成授权
3. 授权成功后自动返回 Healerbook

登录后右上角会显示你的 FFLogs 用户名。

### 首次发布

1. 点击工具栏的 **「共享」** 按钮
2. 点击 **「发布」**
3. 在确认对话框中确认：发布后任何拿到链接的人都能查看，但只有你能编辑
4. 发布成功后会显示分享链接
5. 点击链接旁的 **复制按钮** 把链接发给队友

<!-- 📸 截图组：共享面板三种状态（横向排列）
     截图 A —— 未登录：显示锁图标 + 「需要登录才能共享时间轴」 + 「登录 FFLogs」按钮
     截图 B —— 已登录未发布：显示锁图标 + 「时间轴未共享」 + 「发布」按钮
     截图 C —— 已发布：显示地球图标 + 分享链接 + 复制按钮 + 「发布更新」按钮 -->

### 更新已发布的时间轴

编辑已发布的时间轴后，共享按钮上会出现 **橙色圆点**，表示有未发布的本地修改。

1. 点击 **「共享」**
2. 点击 **「发布更新」**

### 版本冲突怎么办？

如果你在两台设备上同时编辑了同一份时间轴，更新时可能遇到版本冲突。系统会弹出对话框，显示：

- **本地版本**：你当前设备上的版本及编辑时间
- **服务器版本**：另一台设备提交的版本及更新时间

你需要选择：

- **「保留本地版本」**：用你当前的内容覆盖服务器
- **「使用服务器版本」**：放弃本地修改，采用服务器上的版本

<!-- 📸 截图：版本冲突对话框
     内容：ConflictDialog 弹出状态，显示本地版本和服务器版本的时间戳，
     以及「使用服务器版本」和「保留本地版本」两个按钮。 -->

### 队友如何使用你的链接

队友打开分享链接后：

- 可以查看你的完整减伤规划（只读）
- 可以点击 **「在本地创建副本」** 复制一份到自己的浏览器，自由修改

## 9. 如何参考 TOP100 方案

首页底部的 **「TOP100 参考方案」** 展示了各副本 TOP100 rDPS 排行中治疗组合的数据，帮你参考大佬们怎么安排减伤。

3. 选中后，列表只会显示包含这些职业的记录

点击 **橡皮擦图标** 可以清除所有过滤条件。

## 10. 如何从回放模式切换到编辑模式

从 FFLogs 导入的时间轴默认处于 **回放模式**：显示实战数据，时间轴只读。

如果你想基于导入的数据进行修改（调整减伤安排），需要退出回放模式。

### 操作步骤

1. 点击工具栏上蓝色的 **回放模式图标**
2. 在弹出的说明面板中点击 **「解除回放模式」**
3. 在确认对话框中点击 **确认**

<!-- 📸 截图组：退出回放模式流程
     截图 A —— 工具栏上蓝色回放模式图标的位置（用箭头指示）
     截图 B —— 点击后弹出的说明面板，显示回放模式说明文字和「解除回放模式」按钮 -->

### 注意事项

- **此操作不可撤销**。退出回放模式后，每个玩家的实际受伤数据（逐人伤害明细）会被清除
- 伤害事件和技能使用记录会保留，但属性面板将切换为基于模拟计算的预估效果（而非实战数据）
- 退出后撤销/重做历史会被清空

> 建议：如果你想保留原始回放数据作为参考，可以先在首页对这份时间轴创建一个副本，然后只在副本上退出回放模式进行编辑。

## 11. 键盘快捷键速查

| 快捷键                 | 功能                       |
| ---------------------- | -------------------------- |
| `Ctrl + Z`             | 撤销                       |
| `Ctrl + Shift + Z`     | 重做                       |
| `Ctrl + C`             | 复制选中的伤害事件         |
| `Ctrl + V`             | 粘贴伤害事件到光标位置     |
| `Delete` / `Backspace` | 删除选中的事件、技能或注释 |
| `Escape`               | 取消选择 / 关闭弹出框      |
| `Enter`                | 确认输入（标题编辑等）     |
| `Ctrl + Enter`         | 保存注释                   |
| 鼠标滚轮               | 左右平移时间轴             |
| `Ctrl` + 鼠标滚轮      | 上下滚动技能轨道           |
| 双指缩放               | 缩放时间轴                 |

> Mac 用户请将 `Ctrl` 替换为 `Cmd`。
