# 如何过滤时间轴展示

过滤菜单可以临时减少时间轴上显示的伤害事件与减伤技能，让你专注于当前关心的内容（例如只看坦克的死刑处理、只看团减分配）。过滤是纯视觉的，不会影响减伤计算结果，也不会影响导出。

## 入口

工具栏「视图」菜单旁边的 <Filter :size="16" style="display:inline;vertical-align:middle" /> **过滤按钮**，按钮右侧会显示当前生效的预设名，例如「全部」「仅坦克」。

点击按钮展开下拉菜单，选择一个预设即可立即切换。

## 预置选项

| 预设       | 显示的伤害事件     | 显示的技能                                          |
| ---------- | ------------------ | --------------------------------------------------- |
| **全部**   | 全部（AOE + 死刑） | 全部职业的盾类与百分比减伤                          |
| **仅团减** | 仅 AOE             | 全部职业的盾类与百分比减伤                          |
| **仅 DPS** | 仅 AOE             | 仅 DPS 职业（近战 / 远敏 / 法系）的盾类与百分比减伤 |
| **仅坦克** | 全部               | 仅坦克职业的盾类与百分比减伤                        |
| **仅治疗** | 仅 AOE             | 仅治疗职业的盾类与百分比减伤                        |

> "职业"判定基于使用该技能的玩家，而不是技能本身归属的职业。跨角色共享技能（例如"雪仇"4 坦共享）在"仅坦克"下只会显示给坦克玩家。

## 自定义预设

如果预置选项不够用（例如只想看自己职业的某几个技能），可以创建自定义预设。

### 新增预设

1. 点开过滤下拉菜单 → 点击底部 **「自定义…」**
2. 在「管理预设」对话框右上角点击 <Plus :size="16" style="display:inline;vertical-align:middle" /> 图标
3. 在编辑对话框中填写：
   - **预设名称**：1–20 字，用于区分不同预设
   - **伤害事件类型**：AOE 与死刑的开关
   - **技能选择**：按职能（坦克 / 治疗 / DPS）分组，每个职能可以折叠展开；每个职业一行，可点击单个技能 icon 切换，或点击「全选/取消全选」快捷操作
4. 点击 **「保存」**

新建预设时所有选项**默认全选**（等价于「全部」），你可以从完整列表开始取消不关心的项。

### 编辑或删除

在「管理预设」对话框的预设列表中：

- 点击 <Pencil :size="16" style="display:inline;vertical-align:middle" /> 编辑
- 点击 <Trash2 :size="16" style="display:inline;vertical-align:middle" /> 删除（会弹出确认）
- 按住 <GripVertical :size="16" style="display:inline;vertical-align:middle" /> **拖柄**上下拖动可以调整顺序

<!-- 🎬 动图：拖拽排序

### 切换预设

所有自定义预设会出现在过滤下拉菜单中间区域（与预置选项之间有分隔线），直接点击即可切换。

## 提示

- 过滤器选中状态与自定义预设列表都保存在浏览器本地，下次打开时会自动沿用
- 过滤只影响**看到的东西**，被过滤的减伤仍然参与伤害计算 —— 如果你的 AOE 伤害数字和预期不符，切回「全部」检查是不是有技能被过滤隐藏了
- Excel / Souma 导出永远导出**全量**内容，不受当前过滤影响
- 只读模式、回放模式下也可以使用过滤，方便查看他人时间轴时聚焦关心的部分
