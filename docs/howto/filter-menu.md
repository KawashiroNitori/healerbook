# 如何过滤时间轴展示

过滤菜单可以临时减少时间轴上显示的伤害事件与减伤技能，让你专注于当前关心的内容（例如只看坦克的死刑处理、只看团减分配）。过滤是纯视觉的，不会影响减伤计算结果，也不会影响导出。

## 入口

工具栏「视图」菜单旁边的 <Filter :size="16" style="display:inline;vertical-align:middle" /> **过滤按钮**，按钮右侧会显示当前生效的预设名，例如「全部」「仅坦克」。

点击按钮展开下拉菜单，选择一个预设即可立即切换。

## 预置选项

| 预设       | 显示的伤害事件     | 显示的技能                                          |
| ---------- | ------------------ | --------------------------------------------------- |
| **全部**   | 全部（AOE + 死刑） | 全部职业的盾类与百分比减伤                          |
| **仅团减** | 仅 AOE             | 全部职业的盾类与百分比减伤                          |
| **仅 DPS** | 仅 AOE             | 仅 DPS 职业（近战 / 远敏 / 法系）的盾类与百分比减伤 |
| **仅坦克** | 全部               | 仅坦克职业的盾类与百分比减伤                        |
| **仅治疗** | 仅 AOE             | 仅治疗职业的盾类与百分比减伤                        |

## 自定义预设

如果预置选项不够用（例如只想看自己职业的某几个技能），可以创建自定义预设。

### 新增预设

1. 点开过滤下拉菜单 → 点击底部 **「自定义…」**
2. 在「管理预设」对话框右上角点击 <Plus :size="16" style="display:inline;vertical-align:middle" /> 图标
3. 设置你想要显示的伤害事件与技能
4. 点击 **「保存」**

<!-- 📸 截图：编辑预设对话框
     内容：EditPresetDialog 打开状态，顶部是预设名称输入框（带字符计数），
     中间是 AOE/死刑 switch，下方是按职能分组的技能选择网格（坦克/治疗/DPS 可折叠）。 -->

### 编辑或删除

在「管理预设」对话框的预设列表中：

- 点击 <Pencil :size="16" style="display:inline;vertical-align:middle" /> 编辑
- 点击 <Trash2 :size="16" style="display:inline;vertical-align:middle" /> 删除
- 按住 <GripVertical :size="16" style="display:inline;vertical-align:middle" /> 拖柄上下拖动可以调整顺序

<!-- 🎬 动图：拖拽排序
     内容：管理预设对话框中有 3 个自定义预设，按住拖柄把最底下一条拖到最上方。时长 3-4 秒。 -->

### 切换预设

所有自定义预设会出现在过滤下拉菜单中间区域（与预置选项之间有分隔线），直接点击即可切换。
