# 如何放置和调整减伤技能

## 放置技能

**方法一：双击**

- 在目标职业的技能轨道空白处**双击鼠标左键**，在该时间点添加当前选中的减伤技能

**方法二：右键菜单**

- 在技能轨道空白处**右键**，选择 **「添加」**

> 如果提示"该技能与已有技能重叠"，说明同一个技能在 CD 时间内被放了两次，需要调整时间间隔。

## 调整技能时间

直接**拖拽**技能图标左右移动到合适的位置。

## 查看技能详情

**悬浮**鼠标到技能图标上，会显示技能提示，包含：

- 技能名称和图标
- 咏唱时间、复唱时间（CD）
- 效果范围
- 技能效果描述
- 适用职业

**单击**技能图标可以固定提示框，方便阅读。

## 删除技能

- 选中后按 **Delete** 键
- 或右键选择 **「删除」**
