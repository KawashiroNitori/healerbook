# 如何参考 TOP100 方案

首页底部的 **「TOP100 参考方案」** 展示了各副本 TOP100 rDPS 排行中治疗组合的数据，帮你参考大佬们怎么安排减伤。

## 浏览排行

1. 在首页向下滚动到 **「TOP100 参考方案」** 区域
2. 通过顶部的 **标签页** 切换不同版本/副本层级
3. 点击具体副本名称 **展开** 查看排行表
4. 默认显示前 10 条，点击底部的 **「展开全部 X 条」** 查看更多

## 排行表内容

每条记录显示：

| 列        | 说明                         |
| --------- | ---------------------------- |
| #         | 排名                         |
| 治疗组合  | 两名治疗的名字、职业和服务器 |
| 阵容      | 完整 8 人队伍的职业图标      |
| 合计 rDPS | 两名治疗的 rDPS 总和         |
| 时长      | 战斗用时                     |
| 操作      | 「导入」按钮                 |

## 按阵容过滤

如果只想看和你们队伍职业组成相似的方案：

1. 点击列表上方的 **过滤图标**
2. 在弹出的面板中按角色分组选择你们队伍的职业
3. 选中后，列表只会显示包含这些职业的记录

点击 **橡皮擦图标** 可以清除所有过滤条件。

## 一键导入

看到感兴趣的记录？点击该行的 **「导入」** 按钮，会直接打开 FFLogs 导入流程，帮你把那场战斗的减伤方案导入为一份新的时间轴。
