# 如何阅读减伤效果预估

选中一个 **AOE** 类型的伤害事件后，右侧属性面板会显示减伤效果预估。

> 死刑类事件不会显示预估，因为死刑伤害由坦克单独承受，不参与全队减伤计算。面板会提示"死刑不参与团减计算"。

## HP 条

最直观的指标。显示这次 AOE 后全队（非坦克）的大致血量：

- **绿色** = 剩余 HP
- **红色条纹** = 受到的伤害
- 文字格式：`剩余HP / 最大HP (-伤害量)`

## 减伤构成条

HP 条下方的彩色横条展示原始伤害的去向：

| 颜色 | 含义                    | 例子                   |
| ---- | ----------------------- | ---------------------- |
| 蓝色 | 被百分比减伤吃掉的伤害  | 雪仇、节制、牵制的效果 |
| 黄色 | 被盾吸收的伤害          | 圣光幕帘、鼓舞的盾值   |
| 红色 | 实际掉的血              | 你需要奶回来的量       |
| 灰色 | 溢出伤害（超出最大 HP） | 只在伤害致死时出现     |

鼠标悬浮在每个色块上可以看到具体数值和百分比。

## 生效状态列表

面板最底部分两组列出生效的减伤：

- **百分比减伤**：每个减伤 Buff 的图标和减伤率，右侧显示合计减伤
- **盾值**：每个盾技能的图标和数值，右侧显示合计盾值

> 小贴士：减伤计算使用乘算公式。比如两个 10% 减伤叠加的效果是 `1 - (1-0.1) × (1-0.1) = 19%`，而不是简单的 20%。所以多层减伤叠在一起的边际收益会递减。
