# 如何添加和编辑伤害事件

伤害事件代表副本中 boss 的攻击，是减伤规划的基础。

## 添加伤害事件

**方法一：双击**

- 在伤害事件轨道的空白区域**双击鼠标左键**，会弹出添加对话框，时间自动填为你双击位置对应的秒数

**方法二：右键菜单**

- 在伤害事件轨道空白处**右键**，选择 **「添加」**

<!-- 📸 截图：右键菜单

     显示「添加」「粘贴」「添加注释」等选项。 -->

## 填写事件信息

| 字段       | 说明                                                       |
| ---------- | ---------------------------------------------------------- |
| 事件名称   | 给这次攻击起个名字                                         |
| 时间（秒） | 从战斗开始算起的秒数，支持小数                             |
| 伤害值     | 未减伤的原始伤害数值                                       |
| 攻击类型   | **AOE**（全体攻击）或 **死刑**（坦克单吃）                 |
| 伤害类型   | **物理** / **魔法** / **特殊**（影响牵制和昏乱的生效判定） |

## 编辑已有事件

1. **单击**伤害事件方块将其选中
2. 右侧属性面板会显示所有字段，直接修改即可
3. 修改实时生效，不需要额外保存

## 调整事件时间

直接**拖拽**伤害事件方块左右移动。

<!-- 🎬 动图：拖拽伤害事件
     内容：鼠标按住一个蓝色 AOE 方块，左右拖动到新位置后松手。时长 3-4 秒。 -->

## 复制和粘贴事件

- 选中事件后按 **Ctrl + C** 复制
- 按 **Ctrl + V** 粘贴到当前光标位置
- 也可以右键选择 **「复制」** 和 **「粘贴」**

> 复制文本功能（右键→「复制文本」）会把事件信息以文字形式复制到剪贴板，方便发到聊天软件里。

## 删除事件

- 选中后按 **Delete** 或 **Backspace** 键
- 或右键选择 **「删除」**
- 也可以在属性面板右上角点击垃圾桶图标
